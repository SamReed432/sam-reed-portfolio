Project Description:
Project Soundscape was my Senior Capstone project at Tufts, developed with a team of five and sponsored by Bose. The goal was to solve a key problem in spatial audio: virtual sound sources typically only work when a listener is seated in the "sweet spot" between speakers. Our solution used real-time position tracking to dynamically adjust speaker output, maintaining accurate spatial audio as the listener moves, creating an immersive and walkable soundscape. We built a custom web-based sound editor to design these experiences, which exported to a file read by a MATLAB-based audio engine. This engine used live tracker data to render spatial audio in real time across an 8-speaker setup via a digital-to-analog converter.
